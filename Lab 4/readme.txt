JavaScript Quiz Game

# Lab Assignment 4  
A simple JavaScript quiz game that runs entirely in the browser console using ⁠ prompt() ⁠ and ⁠ alert() ⁠.

# Features
•⁠  ⁠Uses an array of objects to store questions and answers
•⁠  ⁠Takes input using prompt()
•⁠  ⁠Normalizes input using toLowerCase() and trim()
•⁠  ⁠Uses loops, conditionals, and functions
•⁠  ⁠Provides immediate feedback
•⁠  ⁠Tracks and displays the final score

# How to Run
1.⁠ ⁠Open any web browser (Chrome, Edge, Firefox, Safari).
2.⁠ ⁠Right-click → Inspect → Console.
3.⁠ ⁠Copy and paste the entire quiz.js code.
4.⁠ ⁠Press Enter.
5.⁠ ⁠Answer all questions in the prompt windows.
