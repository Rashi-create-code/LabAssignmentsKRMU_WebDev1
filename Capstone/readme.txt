Food Delivery Website

This is a simple responsive food delivery website made using HTML5 and CSS3.
The website includes a header, banner section, menu items, categories, restaurant cards, and a contact form.
Flexbox and CSS Grid are used to create the layout, and media queries make the site work on mobile, tablet, and desktop screens.

Features

1.Responsive header and navigation
2.Hero/banner section
3.Menu items displayed in a grid
4.Food categories / restaurant section
5.Contact or order form
6.Footer with basic information

Technologies Used

* HTML5
* CSS3 (Flexbox, Grid, Media Queries)
