Lab2- EXTERNAL CSS STYLING

OVERVIEW

This lab focuses on applying external CSS to enhance the visual presentation and structure of the html file created in Lab 1. The goal is to focus on using different CSS selectors,the box-model,typography, and positioning techniques.

OBJECTIVE

Link and apply an external CSS stylesheet to existing Lab 1 HTML file.
Style elements using tag type, class, and ID selectors.
Customize typography with Google Fonts.
Develop a consistent color theme using HEX or RGB color values.
Apply the CSS box model to control spacing and layout.
Implementing various CSS positioning techniques- Fixed,Static.
Customize button s using border-radius,background color,and padding.
Style tables to display technical skills or tools.


TECHNOLOGIES USED
1) HTML5
2) CSS
3) Google Fonts
